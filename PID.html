PID 调节教学演示：水桶水位控制模型（可调循环时长）

右侧是可交互的“水桶 + 进出水管 + 曲线”动态模型，左侧给出教学目标和操作流程。 时间轴会在达到“循环时长”后自动从 0 s 重新开始一轮；当调整 PID/目标值等参数时，也会从 0 s 开始新的响应。

“全程模式” = 当前一轮 0–T_c（循环时长）；“窗口模式” = 当前一轮内的局部窗口。

一、教学目标

  1. 理解比例 P、积分 I、微分 D 在控制中的物理意义。
  2. 通过“水桶水位”建立对 PID 调节的直观认知。
  3. 观察不同 Kp、Ki、Kd 以及注水/放水速度对响应全过程和局部细节的影响。
  4. 训练根据响应曲线调整 PID 参数的直觉。

二、演示建议流程

1. 只用 P 控制

  1. 设定：Ki = 0，Kd = 0，只保留 Kp。
  2. 目标水位 SP 取 60，点击“开始”。
  3. 用“全程”模式观察一整轮 0–T_c 的响应。

2. 加入积分 I

  1. 调节 Ki，滑块松开时自动从 0 s 重新开始一轮。
  2. 对比不同 Ki 的整轮曲线；用“窗口模式 + 较小窗口长度”看超调细节。

3. 再加入微分 D

  1. 调节 Kd，同样会自动从 0 s 重新开始。
  2. 观察一轮 0–T_c 中超调、振荡、稳态的差异。

4. 调节执行机构能力

  1. 通过“注水速度 / 放水速度”改变最大流量能力。
  2. 对比不同组合下完整一轮的响应，看看是否容易饱和或响应太慢。
提示：如果你想重新做一次“完整试验”，直接拖动参数滑块或循环时长滑块， 松手后时间会自动回到 0 s，并清空上一轮曲线。

三、思考题

  • 在不同循环时长 T_c 下，系统整定是否需要调整？为什么？
  • 当 Ki 或 Kd 调得太大时，多轮循环中有哪些现象是重复出现的？
